DataFlower

Project Overview

This project aims to simplify the process of text annotation, providing a user-friendly platform that enhances the efficiency and accuracy of preparing datasets for machine learning.

Tools and Libraries

Backend: Python | Flask | SQLite
Frontend: HTML | JavaScript
Data Manipulation: Polars | Pandas
Tool: VSCode
