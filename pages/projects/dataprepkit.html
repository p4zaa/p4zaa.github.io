DataPrepKit is a comprehensive toolkit designed to simplify data preprocessing tasks for tabular data using Polars DataFrame library. It includes functionalities for handling duplicates, one-hot encoding, decoding, and text preprocessing specifically for Thai language data.
GitHub
p4zaa/DataPrepKit

TabularPrepKit

  • Duplicate Label Handling:
    Combine labels for each row to handle duplicates efficiently.
  • One-Hot Encoding:
    Encode specified columns using one-hot encoding with flexible options.
  • One-Hot Decoding:
    Decode one-hot encoded columns back to their original labels seamlessly.

ThaiTextPrepKit

  • Thai Text Normalization:
    Normalize Thai text using pythainlp utilities.
  • Custom Dictionary Integration:
    Extend Thai word dictionary with custom words for more accurate tokenization.
  • Named Entity Recognition (NER) Tagging:
    Tag and process Thai text using NER models.
  • Common Word Fixing:
    Correct common typos and variations in Thai text.
  • Text Preprocessing:
    Comprehensive preprocessing for Thai text, including tokenization, stopword removal, and format retention options.
